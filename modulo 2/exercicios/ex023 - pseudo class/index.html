Pseudo-classes

Pseudo classes são classes que alteram o estado de uma classe. Como no exemplo abaixo:

01
02
03

Com a pseudo-classe "hover" é possível mudar o estado de uma classe. Como no exemplo acima que ao passar o mouse por cima dos blocos os blocos acabam mudando de cor.

Existem várias pseudo-classes para serem exploradas e usadas alem da "hover".
